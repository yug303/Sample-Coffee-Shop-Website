# Sample Coffee Shop

**Responsive sample coffee shop website template** built with HTML, CSS, and vanilla JavaScript — includes Home, Full Menu, and Order pages, mobile-friendly navigation, and smooth scrolling.

## Features

- Static, responsive layout (HTML/CSS)
- Mobile-friendly navigation with hamburger menu
- Smooth in-page scrolling and separate Full Menu & Order pages
- Simple demo order form (client-side confirmation)

## Quick start

1. Clone the repo or download the files
2. Open `index.html` in your browser, or run a local static server:

```bash
# Python 3
python -m http.server 8000
# then open http://localhost:8000
```

## Files

- `index.html` — main landing page
- `menu.html` — full menu page
- `order.html` — order page with demo form
- `styles.css` — site styles
- `script.js` — small UI scripts (mobile menu + smooth scroll)
